🏞️ Landing Page de Turismo – Gramado

Projeto de landing page desenvolvido durante estudos de HTML e CSS, com foco em criação de layout responsivo e apresentação de conteúdo turístico.

🚀 Tecnologias

HTML5

CSS3 (seletores avançados, listas, pseudo-classes)

Design de layout baseado em boas práticas de UI

🎨 Sobre o projeto

A página apresenta Gramado como destino turístico, destacando:

Descrição detalhada da cidade

Principais atrações turísticas

Informações práticas para visitantes

Layout organizado e visualmente agradável

Funcionalidades:

Estrutura semântica (<header>, <main>, <section>, <footer>)

Imagens com object-fit e bordas arredondadas

Destaque para informações importantes usando cores (acents)

Listas ordenadas e não ordenadas para organização de conteúdo

🛠️ Melhorias implementadas

Layout responsivo dentro do possível para desktop

Organização do CSS e HTML para melhor manutenção

Estilização com cores, tipografia e espaçamento coerentes

▶️ Como executar
git clone https://github.com/rodrigomscmontenegro/projeto-turismo
cd projeto-turismo

Abra o arquivo index.html no navegador.

🎯 Objetivo

Praticar a construção de páginas web estáticas, interpretação de layouts e organização de conteúdo para criação de landing pages atrativas.
